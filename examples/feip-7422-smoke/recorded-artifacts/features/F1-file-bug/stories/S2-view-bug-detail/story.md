# S2 - View bug detail

**As a** team member  
**I want to** view a bug's detail page at a stable URL  
**So that** I can see the bug's full information and share the URL with others

## Summary

The user can navigate to a bug's detail page via a stable, shareable URL (e.g., `/bugs/42`). The detail page displays the bug's title, description, status, and unique identifier. This URL is the primary way to reference and retrieve a specific bug.

## E2E scope

This is an end-to-end (UI) story: the user directly interacts with the browser to view a bug's details at its stable URL.
